# 백그라운드 영상 녹화 프로그램 사용설명서

`developed by yeohj0710`

창 하나를 골라 녹화하고, 그 창을 뒤로 보내둔 상태에서도 영상과 소리를 저장할 수 있는 윈도우용 프로그램입니다.

## 처음 시작하기

GitHub에서 받은 repo 폴더를 열고 실행 파일을 더블클릭하면 됩니다. 설치 과정은 없습니다.

1. repo를 다운로드한 뒤 압축을 풉니다.
2. 폴더 안의 `백그라운드 영상 녹화 프로그램.exe` 파일을 실행합니다.
3. `사용설명서.html`은 브라우저로 열어볼 수 있는 설명서입니다. 필요할 때 다시 열어보면 됩니다.
4. `프로그램 구성 파일` 폴더는 그대로 둡니다. 이 폴더 안에 실행에 필요한 파일과 개발용 파일이 들어 있습니다.

사용자는 루트의 실행 파일, 설명서, 녹화 폴더만 보면 됩니다. 나머지는 `프로그램 구성 파일` 안에 정리되어 있습니다.

Windows에서 처음 실행할 때 보안 안내가 뜰 수 있습니다. 직접 받은 파일이 맞는지 확인한 뒤 실행하면 됩니다.

## 녹화하는 방법

1. 프로그램을 열면 왼쪽에 현재 열려 있는 창 목록이 보입니다.
2. 녹화하고 싶은 창을 한 번 클릭해서 선택합니다. 예를 들어 Zoom, 브라우저, 강의 플레이어 같은 창을 고르면 됩니다.
3. 오른쪽의 저장 폴더를 확인합니다. 기본값은 압축을 푼 폴더 안의 `녹화 완료된 동영상`입니다.
4. 저장된 영상을 바로 확인하고 싶으면 `열기` 버튼으로 저장 폴더를 열 수 있습니다.
5. `녹화 시작` 버튼을 누릅니다. 녹화가 시작되면 같은 버튼이 `녹화 종료`로 바뀝니다.
6. 녹화 중에는 중간에 멈추지 않고 그대로 이어서 저장됩니다.
7. 다른 작업을 해야 하면 `녹화창 숨기기 (다른 작업 가능)`를 누릅니다. 다시 보고 싶을 때는 같은 버튼의 `녹화창 다시 띄우기`를 누릅니다.
8. 마우스 커서까지 영상에 넣고 싶으면 `마우스 커서도 녹화하기`를 켭니다. 기본값은 꺼짐입니다.
9. 녹화를 끝낼 때는 바뀐 `녹화 종료` 버튼을 누릅니다. 영상과 소리가 합쳐진 뒤 mp4 파일로 저장됩니다.

녹화 파일 이름은 날짜와 시간으로 만들어집니다.

예: `2026-05-09_12-30-05.mp4`

## 소리 녹음

프로그램이 녹화 시작 시 선택한 창의 앱 소리를 우선으로 잡습니다. 이 기능이 지원되지 않는 PC에서는 기본 출력 소리로 대신 녹음합니다.

### 다른 영상을 같이 보고 싶을 때

선택한 앱 소리 녹음이 지원되는 PC에서는 다른 프로그램에서 재생한 소리가 녹화 파일에 섞이지 않는 쪽으로 동작합니다.

앱별 소리 캡처가 지원되지 않는 PC에서는 기본 출력 소리로 대신 녹음합니다. 이 경우에는 내 PC에서 들리는 다른 소리도 같이 들어갈 수 있으니, 녹화 중에는 다른 소리를 잠깐 멈추는 편이 안전합니다.

예를 들어 녹화 대상은 Zoom이고, 다른 작업용 영상은 다른 브라우저나 다른 플레이어에서 재생하면 녹화 파일에는 Zoom 소리만 들어가는 것이 목표입니다.

### 같은 브라우저 안에서는 주의

Chrome처럼 여러 탭과 창이 같은 프로세스 묶음 안에서 돌아가는 앱은, 정확히 창 하나의 소리만 나누기 어려울 수 있습니다. 녹화 대상도 Chrome이고 내가 볼 다른 영상도 Chrome이면 소리가 같이 잡힐 수 있습니다. 이럴 때는 다른 브라우저나 다른 플레이어를 쓰는 편이 안전합니다.

### 녹화 대상 소리를 내가 안 듣고 싶을 때

녹화 대상 앱 자체를 음소거하면 녹음 파일도 조용해질 수 있습니다. 그래서 대상 앱 안의 음소거 버튼, 브라우저 탭 음소거, Windows 앱 볼륨 음소거로 끄는 방식은 추천하지 않습니다.

“내 귀에는 안 들리지만 녹음은 되는 상태”는 앱마다 다르게 동작합니다. 프로그램 안에서 재생 볼륨을 억지로 바꾸면 저장된 영상의 소리 크기와 싱크가 흔들릴 수 있어서, 이 프로그램에서는 볼륨 조절 기능을 넣지 않았습니다.

다른 방법으로 출력 장치를 분리할 수도 있습니다.

1. 녹화할 앱에서 소리가 실제로 나오게 둡니다.
2. `소리 확인` 버튼을 눌러 Windows 앱별 볼륨 설정을 엽니다.
3. 녹화할 앱의 출력 장치를 내가 안 듣는 장치로 바꿉니다.
   예: 모니터 오디오, 안 쓰는 이어폰, 가상 오디오 장치
4. 내가 볼 다른 영상이나 음악은 평소 쓰는 스피커나 이어폰으로 듣습니다.
5. 짧게 테스트 녹화를 해보고 저장된 파일에 소리가 들어가는지 확인합니다.

### 주의할 점

- 녹화할 앱 자체를 음소거하면 녹음 파일도 조용해질 수 있습니다.
- Windows 앱 볼륨에서 해당 앱을 음소거해도 녹음이 안 될 수 있습니다.
- 출력 장치를 바꾸는 것은 음소거와 다릅니다. 앱은 계속 소리를 내고 있어야 합니다.
- 녹화 중에 Windows 기본 출력 장치를 바꾸면 소리가 끊길 수 있습니다. 장치를 바꿔야 한다면 녹화를 새로 시작하는 편이 안전합니다.
- 처음 쓰는 PC에서는 짧게 테스트 녹화를 해보고 저장된 영상에서 소리가 나는지 확인하는 게 좋습니다.

## 저장 폴더

기본 저장 폴더는 실행 파일 옆에 있는 폴더입니다.

`녹화 완료된 동영상`

다른 위치에 저장하고 싶으면 오른쪽의 `변경` 버튼을 눌러 원하는 폴더를 고르면 됩니다. 한 번 바꾼 위치는 다음 실행 때도 기억됩니다.

녹화가 끝난 뒤에는 `열기` 버튼을 누르면 저장 폴더가 바로 열립니다.

저장 폴더가 비어 있거나 사라졌다면 프로그램이 다시 만들거나, 녹화 시작 전에 선택한 위치를 확인합니다.

## 창 조작 버튼

### 녹화창 숨기기 (다른 작업 가능)

선택한 창을 최소화하지 않고 화면 오른쪽 바깥으로 거의 밀어둡니다. 브라우저 영상이 계속 렌더링되도록 아주 얇은 부분만 남겨두고, 다른 작업창을 올리면 거의 보이지 않습니다.

### 녹화창 다시 띄우기

숨겨둔 창을 원래 위치로 다시 가져옵니다. 강의나 회의 내용을 다시 확인하고 싶을 때 사용합니다.

창을 그냥 최소화하면 안 됩니다. Windows 제목 표시줄의 최소화 버튼을 누르지 말고, 꼭 프로그램 안의 `녹화창 숨기기 (다른 작업 가능)` 버튼을 눌러주세요.

정리하면, 녹화 중에 프로그램 안의 버튼으로 숨기거나 다시 띄우는 건 괜찮습니다. 다만 Windows에서 창을 완전히 숨기거나 최소화하면 일부 프로그램은 화면을 더 이상 그리지 않아서 녹화가 검게 나오거나 멈출 수 있습니다.

## 문제 해결

### 녹화 영상이 검게 나와요

- 녹화할 창이 최소화되어 있지 않은지 확인합니다.
- 창을 닫았다가 다시 열었다면 프로그램에서 창 목록을 새로고침하고 다시 선택합니다.
- 일부 프로그램은 자체 정책 때문에 화면 캡처가 막힐 수 있습니다.

### 소리가 안 들어가요

- Zoom이나 브라우저가 음소거되어 있지 않은지 확인합니다.
- Windows 앱 볼륨에서 해당 앱만 음소거되어 있지 않은지 확인합니다.
- 다른 창 소리가 섞이면 녹화 대상과 다른 작업을 같은 브라우저에서 재생하지 말고, 다른 브라우저나 다른 플레이어로 분리합니다.
- 이어폰, 모니터 스피커, 오디오 인터페이스를 바꿨다면 녹화를 종료한 뒤 다시 시작합니다.
- 짧게 10초 정도 테스트 녹화를 한 뒤 저장된 mp4 파일을 재생해봅니다.

### 녹화 파일이 어디 있는지 모르겠어요

기본 위치는 압축을 푼 폴더 안의 `녹화 완료된 동영상`입니다. 프로그램 오른쪽의 저장 폴더 칸에도 현재 저장 위치가 표시됩니다.

### 실행 파일만 옮겨도 되나요?

권장하지 않습니다. `백그라운드 영상 녹화 프로그램.exe`, `사용설명서.html`, `녹화 완료된 동영상` 폴더, `프로그램 구성 파일` 폴더가 같은 위치에 있어야 합니다.

## 사용 전 확인

수업, 회의, 영상은 녹화가 허용된 경우에만 사용해주세요. 녹화 권한이나 공유 범위가 애매하면 먼저 확인하는 것이 좋습니다.

`사용설명서.html` 파일은 인터넷 연결이 없어도 열 수 있습니다.
