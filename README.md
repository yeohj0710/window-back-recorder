# 백그라운드 영상 녹화 프로그램

창 하나를 골라 녹화하고, 그 창을 숨겨둔 상태에서도 영상과 소리를 저장할 수 있는 Windows 프로그램입니다.

**developed by yeohj0710**

## 다운로드

1. 이 페이지 오른쪽 위의 초록색 `Code` 버튼을 누릅니다.
2. `Download ZIP`을 누릅니다.
3. 내려받은 ZIP 파일의 압축을 풉니다.
4. 압축을 푼 폴더 안의 `백그라운드 영상 녹화 프로그램.exe`를 실행합니다.

![GitHub Download ZIP 위치](프로그램%20구성%20파일/github-download-zip.png)

## 폴더 열고 실행하기

1. 압축을 푼 폴더를 엽니다.
2. `백그라운드 영상 녹화 프로그램.exe`를 더블클릭합니다.
3. Windows 보안 안내가 뜨면 직접 내려받은 파일이 맞는지 확인한 뒤 실행합니다.
4. `프로그램 구성 파일` 폴더는 그대로 둡니다. 이 폴더를 지우거나 이름을 바꾸면 실행이 안 될 수 있습니다.
5. `사용설명서.html`은 브라우저로 열어볼 수 있는 자세한 설명서입니다.

설치 과정은 없습니다. 폴더를 받은 뒤 exe를 실행하면 됩니다.

## 녹화 순서

1. 녹화할 창을 먼저 열어둡니다. 예: Zoom, 브라우저, 강의 플레이어
2. 프로그램을 실행합니다.
3. 왼쪽 목록에서 녹화할 창을 클릭합니다. 창이 안 보이면 `새로고침`을 누릅니다.
4. 오른쪽의 저장 폴더를 확인합니다. 기본값은 `녹화 완료된 동영상`입니다.
5. 저장 위치를 바꾸고 싶으면 `변경`을 누릅니다.
6. 마우스 커서까지 녹화하고 싶으면 녹화 시작 전에 `마우스 커서도 녹화하기`를 켭니다. 기본값은 꺼짐입니다.
7. 녹화 중 내 스피커에서 소리가 안 나게 하고 싶으면 `내 스피커 소리 끄고 녹화하기`를 켭니다.
8. `녹화 시작`을 누릅니다. 녹화가 시작되면 같은 버튼이 `녹화 종료`로 바뀝니다.
9. 다른 작업을 해야 하면 `녹화창 숨기기 (다른 작업 가능)`을 누릅니다.
10. 녹화 중인 창을 다시 보고 싶으면 `녹화창 다시 띄우기`를 누릅니다.
11. 끝낼 때는 `녹화 종료`를 누릅니다.
12. 저장 폴더를 바로 확인하고 싶으면 `열기`를 누릅니다.

녹화 중에는 중간에 일시정지하지 않고 한 파일로 이어서 저장됩니다.

## 저장되는 파일

기본 저장 위치는 압축을 푼 폴더 안의 `녹화 완료된 동영상`입니다.

녹화 파일 이름은 날짜와 시간으로 만들어집니다.

```text
녹화 완료된 동영상/
  2026-05-09_12-30-05.mp4
```

녹화가 끝나면 최종 파일은 mp4 하나만 남습니다.

## 소리 녹음

프로그램은 녹화 시작 시 선택한 창의 앱 소리를 우선으로 잡습니다. 이 기능이 지원되지 않는 PC에서는 기본 출력 소리로 대신 녹음합니다.

선택한 앱 소리 녹음이 지원되는 PC에서는 다른 프로그램에서 재생한 소리가 녹화 파일에 섞이지 않는 쪽으로 동작합니다.

앱별 소리 캡처가 지원되지 않는 PC에서는 기본 출력 소리로 대신 녹음합니다. 이 경우에는 내 PC에서 들리는 다른 소리도 같이 들어갈 수 있으니, 녹화 중에는 다른 소리를 잠깐 멈추는 편이 안전합니다.

Chrome처럼 여러 탭과 창이 같은 프로세스 묶음 안에서 돌아가는 앱은, 정확히 창 하나의 소리만 나누기 어려울 수 있습니다. 녹화 대상도 Chrome이고 내가 볼 다른 영상도 Chrome이면 소리가 같이 잡힐 수 있습니다. 이럴 때는 다른 브라우저나 다른 플레이어를 쓰는 편이 안전합니다.

녹화 대상 앱 자체를 음소거하면 녹음 파일도 조용해질 수 있습니다. 대상 앱 안의 음소거 버튼, 브라우저 탭 음소거, Windows 앱 볼륨 음소거로 끄는 방식은 추천하지 않습니다.

### 녹화 대상 소리를 내가 안 듣고 싶을 때

프로그램의 `내 스피커 소리 끄고 녹화하기`를 켜면 대상 앱은 계속 소리를 내는 상태로 두고, Windows 출력 장치의 마지막 단계만 끕니다. 녹화가 끝나거나 옵션을 끄면 원래 소리 상태로 되돌립니다.

앱별 소리 캡처가 지원되는 PC에서는 이 방법이 가장 편합니다. 기본 출력 소리로 녹음되는 PC에서는 장치마다 결과가 다를 수 있으니, 처음에는 10초 정도 짧게 테스트해보는 게 좋습니다.

다른 방법으로 출력 장치를 분리할 수도 있습니다.

1. 녹화할 앱에서 소리가 실제로 나오게 둡니다.
2. 프로그램의 `소리 확인` 버튼을 눌러 Windows 앱별 볼륨 설정을 엽니다.
3. 녹화할 앱의 출력 장치를 내가 안 듣는 장치로 바꿉니다.
   예: 모니터 오디오, 안 쓰는 이어폰, 가상 오디오 장치
4. 내가 볼 다른 영상이나 음악은 평소 쓰는 스피커나 이어폰으로 듣습니다.
5. 짧게 테스트 녹화를 해보고 저장된 파일에 소리가 들어가는지 확인합니다.

## 녹화창 숨기기

`녹화창 숨기기 (다른 작업 가능)`을 누르면 선택한 창을 최소화하지 않고 화면 오른쪽 바깥으로 거의 밀어둡니다. 브라우저 영상이 계속 렌더링되도록 아주 얇은 부분만 남겨두고, 다른 작업창을 올리면 거의 보이지 않습니다.

창을 그냥 최소화하면 안 됩니다. Windows 제목 표시줄의 최소화 버튼을 누르지 말고, 꼭 프로그램 안의 `녹화창 숨기기 (다른 작업 가능)` 버튼을 사용해주세요.

다시 보고 싶을 때는 `녹화창 다시 띄우기`를 누르면 됩니다.

## 폴더 구성

| 항목 | 설명 |
| --- | --- |
| `백그라운드 영상 녹화 프로그램.exe` | 실행 파일입니다. |
| `사용설명서.html` | 브라우저에서 볼 수 있는 자세한 설명서입니다. |
| `녹화 완료된 동영상` | 녹화된 mp4 파일이 저장되는 폴더입니다. |
| `프로그램 구성 파일` | 실행에 필요한 파일입니다. 삭제하거나 이름을 바꾸면 안 됩니다. |

## 문제 해결

### 녹화할 창이 목록에 없어요

- 녹화할 창을 먼저 열고 `새로고침`을 누릅니다.
- 그래도 안 보이면 녹화할 창을 닫았다가 다시 열고, 프로그램에서 다시 새로고침합니다.

### 녹화 영상이 검게 나와요

- 녹화할 창이 최소화되어 있지 않은지 확인합니다.
- 창을 직접 최소화하지 말고 프로그램 안의 `녹화창 숨기기 (다른 작업 가능)` 버튼을 사용합니다.
- 일부 프로그램은 자체 정책 때문에 화면 캡처가 막힐 수 있습니다.

### 소리가 안 들어가요

- Zoom이나 브라우저가 음소거되어 있지 않은지 확인합니다.
- Windows 앱 볼륨에서 해당 앱만 음소거되어 있지 않은지 확인합니다.
- 이어폰, 모니터 스피커, 오디오 인터페이스를 바꿨다면 녹화를 종료한 뒤 다시 시작합니다.
- 처음 쓰는 PC에서는 10초 정도 짧게 테스트 녹화를 해보고 저장된 mp4 파일을 확인합니다.

### 실행 파일만 옮겨도 되나요?

권장하지 않습니다. `백그라운드 영상 녹화 프로그램.exe`, `사용설명서.html`, `녹화 완료된 동영상` 폴더, `프로그램 구성 파일` 폴더가 같은 위치에 있어야 합니다.

## 사용 전 확인

수업, 회의, 영상은 녹화가 허용된 경우에만 사용해주세요. 녹화 권한이나 공유 범위가 애매하면 먼저 확인하는 것이 좋습니다.
