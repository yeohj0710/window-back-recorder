# 백그라운드 영상 녹화 프로그램

`developed by yeohj0710`

원하는 창 하나를 골라서 녹화하는 윈도우용 프로그램입니다. 창을 뒤로 보내두고 다른 작업을 하면서도 녹화는 계속할 수 있습니다.

## 설치와 실행

배포용 zip을 내려받아 압축을 풀고, 맨 바깥에 있는 파일을 실행하면 됩니다.

```text
백그라운드 영상 녹화 프로그램.exe
```

Python이나 FFmpeg를 따로 설치할 필요 없습니다. 필요한 파일은 옆에 있는 `백그라운드 영상 녹화 프로그램_자료` 폴더 안에 같이 들어 있습니다.

자료 폴더는 프로그램 실행에 필요한 파일이 들어 있는 곳이라 지우거나 이름을 바꾸지 말아주세요.

## 사용법

1. 프로그램을 엽니다.
2. 왼쪽 목록에서 녹화할 창을 고릅니다.
3. 오른쪽에서 저장 폴더를 확인합니다.
4. 소리까지 녹음하려면 `기본 출력 녹음` 또는 `출력 장치 녹음` 항목을 선택합니다.
5. `녹화 시작`을 누릅니다.
6. 창을 치워두고 싶으면 `뒤로 보내기`를 누릅니다.
7. 다시 보고 싶으면 `앞으로 보기`를 누릅니다.
8. 녹음되는 소리를 듣고 싶을 때만 `녹음되는 소리 듣기`를 켭니다.
9. 끝나면 `녹화 종료`를 누릅니다.

기본 저장 폴더 이름은 `녹화 완료 영상`입니다. 파일명은 날짜와 시간으로 만들어집니다.

```text
2026-05-09_12-30-05.mp4
```

## 소리가 안 들어갈 때

소리 항목은 Windows에서 실제로 소리가 나가는 출력 장치와 같은 걸 골라야 합니다.

예를 들어 Zoom 소리가 블루투스 이어폰으로 나가고 있으면 `헤드폰`, 모니터 스피커로 나가고 있으면 모니터 이름이 들어간 항목을 고르면 됩니다.

앱 자체를 음소거하거나 Windows 앱 볼륨에서 그 앱을 음소거하면 녹화 파일도 조용해질 수 있습니다.

## 알아둘 점

- 녹화할 창을 최소화하거나 닫으면 화면이 멈추거나 검게 나올 수 있습니다.
- 다른 창에 가려지는 건 보통 괜찮습니다.
- 일부 프로그램은 자체 정책 때문에 화면 캡처가 막힐 수 있습니다.
- 수업, 회의, 영상은 녹화가 허용된 경우에만 사용해주세요.

## 개발자용 빌드

코드를 고친 뒤 exe만 다시 만들려면 아래 명령어를 실행하면 됩니다.

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\build.ps1
```

배포용 zip까지 만들려면 아래 명령어를 실행하면 됩니다. 이 과정은 개발자 PC에서만 필요하고, 실제 사용자는 이 명령어를 실행할 필요 없습니다.

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\package.ps1
```

완성된 zip은 `dist\백그라운드 영상 녹화 프로그램.zip`에 만들어집니다.
