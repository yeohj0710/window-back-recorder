백그라운드 영상 녹화 프로그램 사용설명서

창 하나를 골라 녹화하고, 그 창을 뒤로 보내둔 상태에서도 영상과 소리를 저장할 수 있는 윈도우용 프로그램입니다.

처음 시작하기 녹화하는 방법 소리 설정 저장 폴더 창 조작 문제 해결

처음 시작하기

이 프로그램은 설치 과정이 없습니다. 압축을 풀고 실행 파일을 더블클릭하면 바로 열립니다.

  1. 받은 zip 파일을 원하는 위치에 압축 해제합니다. 바탕화면이나 다운로드 폴더에 풀어도 괜찮습니다.
  2. 백그라운드 영상 녹화 프로그램.exe 파일을 실행합니다.
  3. 옆에 있는 백그라운드 영상 녹화 프로그램_자료 폴더는 그대로 둡니다. 이 폴더 안에 녹화에 필요한 파일이 들어 있습니다.
Python, FFmpeg 같은 프로그램을 따로 설치할 필요가 없습니다. zip 안에 필요한 파일이 같이 들어 있습니다.
Windows에서 처음 실행할 때 보안 안내가 뜰 수 있습니다. 직접 받은 파일이 맞는지 확인한 뒤 실행하면 됩니다.

녹화하는 방법

  1. 프로그램을 열면 왼쪽에 현재 열려 있는 창 목록이 보입니다.
  2. 녹화하고 싶은 창을 한 번 클릭해서 선택합니다. 예를 들어 Zoom, 브라우저, 강의 플레이어 같은 창을 고르면 됩니다.
  3. 오른쪽의 저장 폴더를 확인합니다. 기본값은 동영상\녹화 완료된 동영상입니다.
  4. 소리도 함께 녹음하려면 소리 항목에서 기본 출력 녹음 또는 실제 소리가 나가는 장치를 선택합니다.
  5. 녹화 시작 버튼을 누릅니다.
  6. 녹화가 시작되면 필요한 경우 뒤로 보내기를 눌러 선택한 창을 화면 뒤쪽으로 보낼 수 있습니다.
  7. 녹화를 끝낼 때는 녹화 종료를 누릅니다. 영상과 소리가 합쳐진 뒤 mp4 파일로 저장됩니다.
녹화 파일 이름은 날짜와 시간으로 만들어집니다. 예: 2026-05-09_12-30-05.mp4

소리 설정

소리를 녹음하려면 Windows에서 실제로 소리가 나가는 출력 장치를 골라야 합니다.

보통은 이걸 선택

기본 출력 녹음: ... (추천)으로 표시된 항목을 먼저 선택해보세요. 현재 PC의 기본 스피커나 이어폰을 녹음합니다.

소리가 안 들어가면

Zoom이나 브라우저 소리가 다른 장치로 나가고 있을 수 있습니다. 예를 들어 블루투스 이어폰으로 듣고 있다면 헤드폰이 들어간 항목을 고릅니다.

녹음되는 소리를 내가 듣고 싶을 때

녹음되는 소리 듣기를 켜면 녹음 중인 소리를 내 스피커로 들을 수 있습니다. 조용히 녹음만 하고 싶으면 꺼두면 됩니다.

주의할 점

  • 녹화할 앱 자체를 음소거하면 녹음 파일도 조용해질 수 있습니다.
  • Windows 앱 볼륨에서 해당 앱을 음소거해도 녹음이 안 될 수 있습니다.
  • 소리 항목을 바꿨다면 짧게 테스트 녹화를 해보고 저장된 영상에서 소리가 나는지 확인하는 게 좋습니다.

저장 폴더

처음 실행하면 기본 저장 폴더가 자동으로 만들어집니다.

동영상\녹화 완료된 동영상

다른 위치에 저장하고 싶으면 오른쪽의 변경 버튼을 눌러 원하는 폴더를 고르면 됩니다. 한 번 바꾼 위치는 다음 실행 때도 기억됩니다.

저장 폴더가 비어 있거나 사라졌다면 프로그램이 다시 만들거나, 녹화 시작 전에 선택한 위치를 확인합니다.

창 조작 버튼

뒤로 보내기

선택한 창을 다른 창 뒤쪽으로 보냅니다. 다른 작업을 하면서 녹화를 계속하고 싶을 때 사용합니다.

앞으로 보기

뒤로 보낸 창을 다시 앞으로 가져옵니다. 강의나 회의 내용을 다시 확인하고 싶을 때 사용합니다.

작게 두기

선택한 창을 화면 구석에 작게 둡니다. 완전히 뒤로 보내기 전 상태를 확인할 때 좋습니다.

크기 복원

작게 둔 창을 이전 크기로 되돌립니다.

창을 최소화하거나 닫으면 화면이 멈추거나 검게 나올 수 있습니다. 다른 창에 가려지는 것은 괜찮지만, 최소화는 피하는 편이 좋습니다.

문제 해결

녹화 영상이 검게 나와요

  • 녹화할 창이 최소화되어 있지 않은지 확인합니다.
  • 창을 닫았다가 다시 열었다면 프로그램에서 창 목록을 새로고침하고 다시 선택합니다.
  • 일부 프로그램은 자체 정책 때문에 화면 캡처가 막힐 수 있습니다.

소리가 안 들어가요

  • 소리 항목에서 기본 출력 녹음을 먼저 선택합니다.
  • 이어폰, 모니터 스피커, 오디오 인터페이스를 쓰고 있다면 그 장치 이름이 들어간 항목을 선택합니다.
  • Zoom이나 브라우저가 음소거되어 있지 않은지 확인합니다.
  • 짧게 10초 정도 테스트 녹화를 한 뒤 저장된 mp4 파일을 재생해봅니다.

녹화 파일이 어디 있는지 모르겠어요

기본 위치는 동영상\녹화 완료된 동영상입니다. 프로그램 오른쪽의 저장 폴더 칸에도 현재 저장 위치가 표시됩니다.

실행 파일만 옮겨도 되나요?

권장하지 않습니다. 백그라운드 영상 녹화 프로그램.exe와 백그라운드 영상 녹화 프로그램_자료 폴더가 같은 위치에 있어야 합니다.

사용 전 확인

수업, 회의, 영상은 녹화가 허용된 경우에만 사용해주세요. 녹화 권한이나 공유 범위가 애매하면 먼저 확인하는 것이 좋습니다.

이 문서는 프로그램 폴더 안의 사용설명서.html 파일입니다. 인터넷 연결이 없어도 열 수 있습니다.
