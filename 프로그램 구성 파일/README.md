# 백그라운드 영상 녹화 프로그램

`developed by yeohj0710`

GitHub repo를 다운로드한 뒤, 루트의 실행 파일을 바로 열면 됩니다.

```text
백그라운드 영상 녹화 프로그램.exe
```

루트 폴더는 사용자에게 필요한 항목만 보이도록 정리되어 있습니다.

```text
백그라운드 영상 녹화 프로그램.exe
사용설명서.html
녹화 완료된 동영상\
프로그램 구성 파일\
```

`프로그램 구성 파일` 폴더에는 실행에 필요한 파일과 개발용 파일이 들어 있습니다. 사용자는 보통 열어볼 필요 없습니다.

소리는 녹화 시작 시 기본 출력으로 자동 녹음됩니다. 사용자가 따로 장치를 고를 필요는 없습니다.

자세한 사용법은 `사용설명서.html`을 열어보면 됩니다.
